What is CSS?

CSS stands for Cascading Style Sheets. It is a style sheet language which is used to describe the look and formatting of a document written in markup language. It provides an additional feature to HTML. It is generally used with HTML to change the style of web pages and user interfaces.

Types of CSS

  • Inline CSS
  • Internal CSS
  • External CSS

  1. Inline CSS

Inline CSS contains the CSS property in the body section attached with element is known as inline CSS. This kind of style is specified within an HTML tag using the style attribute.

What is ID?

The difference between class and ID in HTML is that IDs are unique — classes are not. In practice, this means that multiple elements on a page can have the class, while elements can have only one ID and each page can have only one element with that ID.
